# Project Brief: GenericQueue Web Interface

The goal of this project is to provide a web-based interface for the existing `GenericQueue` application.

## Core Requirements
- Maintain compatibility with the existing SQL Server database schema.
- Replicate the dynamic, XML-driven form generation logic in a web environment.
- Support "Types" of queues and their associated workflows.
- Enable viewing, editing, and saving item details via the web.

## Goals
- Transition from a thick-client WPF application to a modern web interface.
- Provide a responsive and accessible UI.
- Ensure easy deployment and multi-user access.
